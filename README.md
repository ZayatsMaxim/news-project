# Новости

Приложение для просмотра и редактирования постов. Данные загружаются с [DummyJSON](https://dummyjson.com).

## Возможности

- **Список постов** — реализован в виде grid с карточками. Реализована пагинация
- **Поиск** по заголовку, тексту или ID автора
- **Детали поста** — модальное окно с информацией об авторе, комментариями, реакциями и просмотрами
- **Редактирование** — изменение заголовка и текста поста
- **Навигация** — переход к предыдущему/следующему посту из модалки

## Стек

- **Vue 3** (Composition API, `<script setup>`)
- **TypeScript**
- **Pinia** — состояние (список постов, детали поста)
- **Vuetify 3** — UI
- **Vite** — сборка и dev-сервер
- **Vitest** — юнит-тесты

## Требования

- Node.js: `^20.19.0` или `>=22.12.0`

## Установка и запуск

```bash
npm install
npm run dev
```

Приложение откроется по адресу из вывода Vite (обычно `http://localhost:5173`).

## Переменные окружения

- `VITE_API_BASE_URL` — базовый URL API (по умолчанию в проекте задан под DummyJSON `https://dummyjson.com`). При необходимости создайте `.env` и задайте переменную.

## Структура проекта

- `src/components/posts/` — список постов, карточка, модалка деталей
- `src/stores/` — Pinia-сторы (список постов, детали поста)
- `src/api/` — запросы к API, маппинг, поиск по постам
- `src/composables/` — координатор постов, снэкбар ошибок
- `src/dto/` — типы данных постов, комментариев, пользователей
- `src/__tests__/` — юнит-тесты
